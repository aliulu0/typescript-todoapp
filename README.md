# Setup

1. Clone the repository and install it's features.

```
npm install or yarn install
```

2. Use the project locally(http://localhost:3000).

```
npm start or yarn start
```

## Project Features

- Create Todo <br />
- Delete Todo <br />
- Edit Todo's title <br />
- Change todo's completed propertie <br />
- Clear all todos <br />
